# Project Overview

## Purpose
This is an experimental intelligent agent for a Pi Bramble (Raspberry Pi cluster). The project is in its very early stages with minimal functionality.

## Tech Stack
- **Language**: Python 3.11+
- **Package Management**: Standard Python packaging with pyproject.toml
- **No external dependencies**: Currently uses only Python standard library
- **License**: Public domain (Unlicense)

## Current State
- Single entry point in `main.py` with basic "Hello from agent!" output
- Minimal project structure with no additional modules or packages
- No testing framework or dependencies set up yet
- No CI/CD or automation scripts present
